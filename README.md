# JP Auto Detailing Website

Marketing website for JP Auto Detail built with Next.js App Router, React, and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` - start local development server
- `npm run lint` - run ESLint
- `npm run build` - create production build
- `npm run start` - start production server

## Tech stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript
